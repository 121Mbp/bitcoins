## 소개
코인들의 시가총액 및 현재 가격 등의 정보와 24h 동안의 가격 변동에 대해 차트로 표현했으며, 많은 코인들의 리스트를 무한 스크롤로 구현했습니다. 검색, 코인 정보 등 업데이트 예정입니다.

## 기능
- 코인 정보 리스트
  
## 링크
https://bitcoins-sage.vercel.app/

## 실행
```bash
$ npm i 
```
